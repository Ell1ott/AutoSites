SHEET A-03 / SCHEMA

Working
process, in section

Drawing 03/06
Axonometric · scale 1:25
Six stages, left to right
01 · A
Brief

A long conversation, usually two of them. We listen for the actual problem, the budget that already exists, and the one constraint nobody has named yet.

02 · B
Schema

Sitemap, content model, type system, two reference compositions. The deliverable is a five-page document you could hand to any other studio and continue.

03 · C
Scaffold

Repository, build pipeline, content lake, accessibility ground rules. We commit a deployable empty site within a working week.

04 · D
Joinery

Pages, components, edges. We write CSS the way a joiner cuts a tenon — measure twice, name everything, leave a margin for the wood to move.

05 · E
Finish

Performance, copy pass, image pipeline, fallback fonts, schema markup, the small print of the build that decides whether it ages well.

06 · F
Tend

A retainer measured in hours, not tickets. Quarterly review of analytics, content, and dependencies. We sweep the workshop on the first Friday of each month.

Disciplines —

DISC. 01 / DEV BENCH A

Bespoke Development

— hand-cut websites
ScopeMarketing sites, longform editorial, small applications. One studio, one site at a time.
Duration6 to 12 weeks, including review windows. Larger systems quoted in phases.
DeliverablesRepository, deploy pipeline, component library, content guidelines, two months of post-launch tending.
ToolsAstro, Eleventy, Next, Remix. Plain CSS. No drag-and-drop builders.
OutputStatic or hybrid HTML; ~ 60–90 KB per page; first paint under 1.0 s on 3G.
DISC. 02 / CMS BENCH B

Adaptive CMS

— content models that fit you
ScopeEditorial structures, archives, multi-author publishing, internationalised catalogues.
Duration4 to 10 weeks. Often a second pass three months in, once writers have used it.
DeliverablesSchema, editor UI, preview build, content seed, hand-off PDF written for one specific person.
ToolsSanity, Payload, TinaCMS, Decap. Sometimes a custom flat-file approach.
OutputTyped schemas, JSON, markdown; auditable migrations; export-friendly.
DISC. 03 / SEO BENCH C

Organic SEO

— foundations, not tactics
ScopeInformation architecture, semantic markup, content audit, link health, page-experience metrics.
DurationAn initial sprint of 3 weeks, then a quarterly cadence for at least a year.
DeliverablesKeyword landscape map, technical audit, redirect plan, content briefs, dashboard you can read in two minutes.
ToolsAhrefs, Search Console, Plausible, Schema.org, a notebook.
OutputMarkdown reports, JSON-LD, redirect maps; nothing locked behind a vendor.
SHEET A-05 / TOOLCHAIN

Stack —
what we actually use

Last revised 04.2026
Subject to taste

NODE AEDITOR
Neovim

A modal editor with no theme. Two plugins: telescope and treesitter. The point is fewer things to look at, not more.

NODE BLANGUAGES
TS · CSS · MDX

Strict TypeScript for application code, plain CSS with custom properties, MDX for anything that wants to read like prose.

NODE CCMS
Sanity / Payload / Flat

Three options chosen by the team writing the content. Hosted, self-hosted, or a folder of markdown — never the same on every project.

NODE DHOSTING
Cloudflare · Vercel

Edge-served static where it suits, server-rendered where it must. We avoid hosting that ties you to a runtime you cannot leave.

NODE ETYPE
Commercial + Self-hosted

Foundries we trust: Klim, Grilli, Production Type, Pangram. Self-hosted via WOFF2; subset to the language pairs in use.

NODE FMONITORING
Plausible · Sentry

Privacy-first analytics, transparent dashboards, error reporting that respects user data. No third-party scripts you cannot read.

N. 0012024.03.18

A page is a room. If a visitor cannot picture standing inside it after one second, the door is wrong, not the wallpaper.

N. 0022024.05.02

Most performance work is removal. Add nothing for a week and measure what you lose. Often nothing.

N. 0032024.06.27

A grid is a contract with the reader. Break it on purpose, three times per page maximum, never to be clever.

N. 0042024.09.11

CMS schemas should be designed by the person who will type into them on a Tuesday afternoon. Anything else is cosplay.

N. 0052024.11.04

If a component name needs an adjective, it is two components pretending to be one. Split the joint.

N. 0062025.01.22

SEO is mostly a question of being honest about what the page is. The rest is plumbing — sitemap, headings, redirects, patience.

N. 0072025.03.10

Hover states are a luxury for desktop visitors. Do not encode meaning there. Touch first, mouse second.

N. 0082025.06.15

Type is a temperature. Inter at 14/22 reads cooler than 14/20 — by enough that you notice the room change.

N. 0092025.09.07

A hand-off document of two pages is read. A hand-off of twenty is filed. Choose the cost you want to pay later.

N. 0102026.02.19

When in doubt: hairline, monospace label, ten pixels of breathing room. The drawing will explain itself.
